[[def: pre-rotation,pre-rotation, pre-rotated]]

~ A key rotation mechanism whereby a set of rotation keys are pre-commited
~    using cryptographic digests, as defined by the [KERI specification](https://trustoverip.github.io/kswg-keri-specification/).

~ More detail:
~    [WebOfTrust glossary](https://weboftrust.github.io/WOT-terms/docs/glossary/pre-rotation).
